# Eleven — macOS releases

Public download and auto-update channel for the **Eleven** macOS app.

- **Latest DMG:** see [Releases](../../releases/latest).
- Auto-updates are delivered via [Sparkle](https://sparkle-project.org); the app reads `appcast.xml` from the latest release.

The app source lives in a private repository; only signed, notarized release artifacts are published here.
